    Download and Installation

    Letswave7 can be downloaded as an archive from the NOCIONS Github repository.

    Download

    You can download the package directly of Letswave7 from Github or the mirror site. Or you can visit the NOCIONS Github repository and download the zip file.

    Installation

    1. Unzip the zip archive of letswave7 in the folder of your choice.
    2. Launch Matlab
    3. In the Matlab menu ribbon, click on “Set Path”.
    4. In the Set Path window, click button “Add Folder”.
    5. Select the location of the Letswave7 folder
    6. Save the modified path and close.

    To check whether the install is successful or not, input “letswave7” in the commond windows of Matlab. The pop-up of manager module, the main interface, indicates that the installation is successful.

    Note: Letswave7 could NOT be used simutanously with eeglab, fieldtrip and the old version of Letswave. Once you used the letswave7, the path of eeglab, fieldtrip and etc. will be temporarily removed. If you want to use these toolboxs again, please restart the matlab or open a new matlab. Letswav6 can be downloaded here.
