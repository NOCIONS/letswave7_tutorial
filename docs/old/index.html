    Welcome to Letswave

    Letswave is a free software toolbox to analyze electroencephalographic (EEG) and other neurophysiological signals. This current version Letswave 7 has been rewritten entirely, taking into consideration structural limitations of Letswave 6. As compared to existing signal processing toolboxes, emphasis is placed on intuitive and streamlined tools to process and visualize EEG data, with a shallow learning curve. The new version provides advanced scripting possibilities.

    Authors and Contributors

    The project is managed by André Mouraux (Institute of Neuroscience, Université catholique de Louvain, Belgium), in collaboration with Gan Huang and Bruno Rossion (Institute of Neuroscience, Université catholique de Louvain), Li Hu (Southwest University, China) and Giandomenico Iannetti (University College London, UK). It borrows a number of functions from the Fieldtrip toolbox (F.C. Donders Institute), and EEGLAB (Swartz Center for Computational Neuroscience).

    Download and setup

    Letswave 7 runs on Matlab versions 2012a under Windows, Mac OSX, and Linux or Unix (Matlab 2015b recommended). It is recommended to use Letswave 7 on Matlab 2015b or later with 8 Gb RAM or more. The use of Letswave 7 depends on Matlab toolbox of Signal Processing, Wavelet, and Statistics and Machine Learning for the corresponding function filtering, wavelet transform and statistic analysis.

    Download
    You can download the package directly from Github or the mirror site.
    The old version Letswave 6 can be download here.

    Setup
    Unzip the file you download into the folder of your choice. Add the folder into the the path of Matlab.
    You can use it by input "letswave7" in the commond windows of Matlab.

    Supported data formats

  • EEGLAB
  • ANT Neuro, eeprobe/cnt-riff
  • NeuroScan
  • Biosemi BDF
  • BrainVision
  • Electrical Geodesics, Inc. (EGI)
  • Generic standard formats

    • Alternatively, you can also import data as Matlab MAT files or exported ASCII files. If you are unable to import your data format, submit an issue here.
